##  Projets

###  Analyse factorielle des données du Titanic
**Analyse Factorielle des Correspondances Multiples (AFCM)** appliquée aux données du naufrage du Titanic afin d'étudier les relations entre les caractéristiques socio-démographiques des passagers (classe, sexe, âge) et la survie.

**Objectifs :**
- Étudier les associations entre variables qualitatives
- Identifier les structures sous-jacentes aux profils de passagers
- Interpréter les inégalités sociales liées à la survie

**Méthodes et outils :**
- R / RMarkdown  
- Analyse descriptive  
- Test du χ², V de Cramér  
- AFCM (FactoMineR, factoextra)
